# Installation - SuBBoard

1 - Installer node : https://nodejs.org/en/download/ \n
2 - Vérifier l'installation avec la commande "node -v" \n
3 - Installer MongoDB : https://www.mongodb.com/download-center#community \n
4 - Lancer la commande "mongod" \n
5 - Lancer la commande npm install dans le dossier SuBBoard ainsi que dans le dossier client à l'intérieur. \n
6 - A l'aide de 2 terminal exécuter la commande "npm start" dans le dossier SuBBoard ainsi que dans le dossier client à l'intérieur. \n \n

Le site web devrait se lancer à l'adresse : http://localhost:3000/
